📋 Firebase Rules — Kích hoạt lần đầu

Để web tự kích hoạt tài khoản lần đầu, cần cho phép write 1 lần:

{ "rules": { "Library": { ".read": true }, "SystemData": { ".read": true }, "Users": { "$uid": { ".read": true, ".write": true } }, "Projects": { "$uid": { ".read": true } }, "Estimates": { "$uid": { ".read": true } }, ".read": false, ".write": false } }

⚠️ Sau khi tất cả user đã đăng nhập lần đầu → đổi ".write": false lại để bảo mật!
