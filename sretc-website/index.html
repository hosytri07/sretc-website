Câu hỏi thường gặp

Giải đáp nhanh trước khi dùng thử

Phần mềm có miễn phí không?
Có — S-RETC Toolkit hoàn toàn miễn phí cho kỹ sư thiết kế cầu đường. Một số tính năng AI yêu cầu API Key từ Groq (cũng miễn phí với quota hàng ngày).
Cần cài thêm phần mềm gì không?
Chỉ cần .NET Framework 4.7.2 (Windows 10/11 đã có sẵn, Windows 7 cần cài thêm — bộ cài đặt sẽ tự kiểm tra và hướng dẫn). Không cần AutoCAD để dùng các module khác, chỉ cần AutoCAD nếu muốn dùng module CAD tự động.
Dữ liệu của tôi được lưu ở đâu?
Dữ liệu dự án, dự toán được lưu trên Firebase Realtime Database (Google) — an toàn, có thể truy cập từ Web Companion bất kỳ lúc nào. Dữ liệu cục bộ (file xuất Excel, Word) lưu trên máy tính của bạn.
Web Companion khác gì Desktop App?
Desktop App có đầy đủ tính năng: CAD tự động, OCR, xuất Excel/Word, dự toán.
Web Companion dùng để xem dự án, chatbot AI, tra cứu tài liệu — truy cập từ điện thoại hoặc bất kỳ thiết bị nào có trình duyệt.
Module CAD có cần AutoCAD bản quyền không?
Cần AutoCAD (bất kỳ phiên bản nào từ 2015 trở lên). S-RETC tạo file .scr script — chạy trong AutoCAD bằng lệnh SCRIPT. Không yêu cầu AutoCAD bản quyền để sử dụng các module khác.
